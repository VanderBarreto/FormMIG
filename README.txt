Sistema operacional Linux


ATIVAR REST

copie a pasta "rest" localizado na pasta de arquviso base do DSpace, na pasta "webapps", para a pasta "webapps" do diretório do apache-tomcat

# cp -r /DIRETÓRIO BASE DO DSPACE/webapps/rest /DIRETÓRIO BASE DO TOMCAT/apache-tomcat/webapps/

Edite o ar
# vim /DIRETÓRIO BASE DO TOMCAT/apache-tomcat/webapps/rest/WEB-INF/server.xml


INSTALAR FORMULÁRIO

Instalação do python e bibliotecas requeridas utilizando o Miniconda

Baixar o Miniconda

# wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh

execute o comando

# bash Miniconda3-latest-Linux-x86_64.sh
